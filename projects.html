Computer Programming
How does the brain represent computer programs? Following up on work by Ivanova et al. (2020), who showed that comprehending programming and natural languages utilize different neural resources, we set out to investigate the properties encoded in human and artificial neural representations of computer code. A group of 24 participants were instructed to calculate the output of short python programs during an fMRI session. These programs varied in content (math calculation vs. string manipulation), structure (sequential vs. iterative vs. conditional), and various continuous static AST-related (e.g. # of nodes) and dynamic (e.g. runtime steps) properties. We extracted voxel pattern responses during program comprehension and problem solving, and evaluated the performance of these representations on downstream code property benchmarks using a linear decoder. In parallel, we passed these same programs to computational language models of code, ranging from BOW to SOTA transformers, and probed the resulting embeddings for the same code properties. Finally, we investigated affine mappings between the multivariate representations of these programs in functionally localized brain networks and in code models, to evaluate and compare their learned representations. We found that several brain networks and code models encode specific aspects of code syntax and semantics, and that these systems share linearly decodable information across their learned representations. Detailed analyses and results are available in the following paper and code base.

Manuscripts:
- Srikant S*, Lipkin B*, Ivanova A, Fedorenko E, O'Reilly UM. (under review). Representations of Computer Programs in the Human Brain. Preprint. GitHub.

Functional Markers & Atlases
What are the topographies of the brain's functional networks, e.g. language, multiple demand (MD), theory of mind (ToM), default-mode (DMN), etc.? Are individual differences in network effect size and spatial structure stable and reliable over time? We leverage data from 800+ subjects collected over 15+ years to functionally localize key brain networks on an individual level. We then evaluate stability and reliability of ROI-level network response strength, spatial topography, lateralization, and other key markers on both short (between runs within a session) and long (between sessions; spanning up to years) time-scales, offering strong support in defense of individual-level neural markers. Additionally, we aggregate individual network responses to build probabilistic atlases of each network topography, providing a valuable tool for effect localization in future studies.

Language Network Characterization
What is the relationship between LH & RH language regions? How do the basal temporal language area (BTLA), Cerebellum, & Hippocampus contribute to language? We leverage data from 1000+ sessions, and 80+ experiments tiling specific features of language and other hierarchically structured processes to provide a specific and nuanced functional profile for lateralization within the language network, and the contribution of external modules to the processing goals of this network.

Manuscripts:
- Small H*, Lipkin B*, Affourtit J, Pongos A, Fedorenko E. (in prep.). The right-hemisphere language network is less functionally specialized for language processing, but only for select domains.
- Lipkin B*, Small H*, Siegelman M, Ziegler J, Paunov A, Fedorenko E. (in prep.). Functional characterization of the high-level language-responsive area in left ventral temporal cortex. 



BrangLab - David Brang - University of Michigan

Glioma-infiltrated Cortex Characterization
To what extent is glioma-infiltrated cortex functional? Recent work, e.g. Venkatesh et al. (2020) has demonstrated that glioma cells interact with neurons and modulate local circuitry. However, it is unclear the extent to which these cells participate in meaningful computations. We utilize intracranial recording (iEEG; ECoG) to sample both normal-appearing and glioma-infiltrated cortex during a speech task. We find that glioma-infiltrated cortex engages in synchronous activity during task performance in a manner similar to normal-appearing cortex; however, these representations from glioma-infiltrated cortex fail to decode nuanced features of speech production, e.g. mono- vs. poly-syllabic utterance, while these features are robustly decodable from healthy-appearing cortex. Relatedly, we show that signals from glioma-infiltrated cortex have decreased entropy, which may affect its ability to encode information.

Manuscripts:
Aabedi A*, Lipkin B*, Kaur J, Kakaizada S, Reihl S, Young JS, Lee AT, Krishna S, Chang EF, Brang D, Hervey-Jumper SL. (2021). Functional alterations in cortical processing of speech in glioma-infiltrated cortex. PNAS, 118(46). PDF.

Side Projects
As part of my independent learning, I am also frequently working on side projects and building useful tools. Here's an example:

OptSent: a python library for efficiently solving shortest-path problems over sets of strings given any arbitrary transition objective. This package has primarily been used to generate psycholinguistic stimulus materials, where it is often desirable to minimize compositionality between adjacent trials, operationalized as a function of the joint causal language modeling probability.
